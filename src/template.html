Smart Brain

'Smart brain' is a full-stack web app that uses Clarifai API to locate a face in a picture uploaded by a user via URL. User needs to register first and can sign in back to the app later with correct information. Front end is built in React and backend is built with nodeJS and Express. Database is in PostgreSQL and the app is deployed on Heroku. User's entry count is updated and stored in the backend database.

Now, let's register a new user and play with it! [Ps: It may take some time to load :)]
